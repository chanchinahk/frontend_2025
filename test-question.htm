<span>


A <span> element which is used to color a part of a text:
Definition and Usage
The <span> tag is an inline container used to mark up a part of a text, or a part of a document. The <span> tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The <span> is much like the <div> element, but <div> is a block-level element and <span> is an inline element.


Font Weight

In CSS, the maximum font weight is 900. This value represents the heaviest, or boldest, weight available for most fonts. The font-weight property allows you to specify the thickness of text, with values ranging from 100 (lightest) to 900 (heaviest), in increments of 100. Keywords like "normal" (400) and "bold" (700) are also used, but font-weight: 900 provides the maximum visual weight.

Here's a more detailed explanation:

Numeric Values:
The font-weight property accepts numeric values from 100 to 900, corresponding to different levels of boldness.

Maximum Weight:
900 is the highest numeric value and represents the thickest possible stroke for most fonts.

Keywords:
"normal" is equivalent to 400, and "bold" is equivalent to 700. "bolder" and "lighter" keywords adjust the weight relative to the parent element's weight.

Variable Fonts:
Variable fonts can have even finer control over font weight, potentially allowing values beyond 900 (e.g., up to 1000), but 900 is the maximum for standard, non-variable fonts.
